# ShardDB Motion Canvas

This directory contains the Motion Canvas animation project for the ShardDB
system explainer.

## Quick Start

```bash
cd animation
npm install
npm start
```

The editor should open at [http://localhost:9000/](http://localhost:9000/).

## Current Scope

- project skeleton
- shared theme and topology data
- reusable visual components
- `S01_Overview` scene
